+# Alina Finance Bot
+
+Telegram-бот для учёта личных финансов с живым тоном общения.
+
+## Запуск (локально / Bothost)
+
+1. Установите зависимости:
+   ```bash
+   pip install -r requirements.txt
+   ```
+2. Настройте переменные окружения:
+   - `TELEGRAM_BOT_TOKEN` — токен бота (обязательно)
+   - `ADMIN_ID` — Telegram ID админа (опционально, по умолчанию `463620997`)
+   - `BOT_DB_PATH` — путь до SQLite файла (опционально, `bot_data.sqlite3`)
+   - `SUPPORT_CONTACT` — контакт для доступа (опционально, `@privetetoalina`)
+   
+   Можно создать файл `.env` рядом с `bot.py`, например:
+   ```env
+   TELEGRAM_BOT_TOKEN=123:abc
+   ADMIN_ID=463620997
+   BOT_DB_PATH=bot_data.sqlite3
+   SUPPORT_CONTACT=@privetetoalina
+   ```
+3. Запустите:
+   ```bash
+   python bot.py
+   ```
+
+## Почему SQLite
+
+В проекте выбрана SQLite вместо JSON-файлов, потому что:
+- лучше переносит конкурентные записи;
+- не теряет структуру при частично битых файлах;
+- удобнее для миграций и будущих аналитических запросов;
+- не требует отдельного сервера БД.
+
+## Безопасность
+
+Если токен уже публиковался в чате/репозитории — обязательно перевыпустите его через BotFather и замените в `.env`.
+
+## Архитектура
+
+- `bot.py` — маршрутизация команд, UI-кнопки, сценарии диалога.
+- `storage.py` — слой SQLite и работа с пользователями/состоянием.
+- `phrases.py` — тональность, большие банки фраз, анти-повторы.
+- `analytics.py` — поведенческие эвристики и инсайты.
+- `config.py` — конфигурация окружения.
+
